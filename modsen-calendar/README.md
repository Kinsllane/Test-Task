# 📅 Modsen Calendar

Современное веб-приложение календаря с полным функционалом управления событиями, аутентификацией пользователей и адаптивным дизайном.

## 🚀 Основные возможности

### 👤 Система аутентификации
- **Регистрация и вход** через Firebase Authentication
- **Валидация форм** с детальными сообщениями об ошибках
- **Управление профилем** - редактирование имени, фамилии и аватара
- **Безопасный выход** из системы

### 📋 Управление событиями
- **Создание событий** с полной информацией (название, место, дата, время, описание, цвет)
- **Редактирование событий** с сохранением в реальном времени
- **Удаление событий** с подтверждением
- **Drag & Drop** - перетаскивание событий между датами
- **Цветовая категоризация** - 8 предустановленных цветов для событий

### 🗓️ Виды календаря
- **Дневной вид** - детальный просмотр одного дня
- **Недельный вид** - обзор недели с почасовой сеткой
- **Навигация** - переключение между неделями и датами
- **Адаптивный дизайн** - оптимизация для мобильных устройств

### ⭐ Система избранного
- **Добавление в избранное** - отмечайте важные события
- **Список избранных** - быстрый доступ к важным событиям
- **Визуальные индикаторы** - звездочки для избранных событий

### 🔍 Поиск событий
- **Интеграция с KudaGo API** - поиск реальных событий Санкт-Петербурга
- **Автодополнение** - предложения при вводе названия события
- **Debounce поиск** - оптимизированные запросы к API
- **Обработка ошибок** - graceful fallback при недоступности API

### 📱 Адаптивность и UX
- **Полностью адаптивный дизайн** - от мобильных до десктопных экранов
- **Touch-friendly интерфейс** - оптимизация для сенсорных экранов
- **Плавные анимации** - современные CSS transitions
- **Toast уведомления** - информативная обратная связь
- **Загрузочные состояния** - индикаторы процесса загрузки

### 🔔 Уведомления
- **Напоминания о событиях** - уведомления за 15 минут до начала
- **Browser notifications** - нативные уведомления браузера
- **Автоматическая проверка** - фоновый мониторинг предстоящих событий

## 🛠️ Технологический стек

### Frontend
- **React 18** - современная библиотека для UI
- **TypeScript** - строгая типизация для надежности кода
- **Redux Toolkit** - предсказуемое управление состоянием
- **React Router** - клиентская маршрутизация
- **SCSS Modules** - модульные стили с препроцессором

### Backend & Services
- **Firebase Authentication** - безопасная аутентификация
- **Firebase Firestore** - облачная база данных (готова к интеграции)
- **KudaGo API** - внешний API для поиска событий
- **LocalStorage** - локальное хранение данных

### Development Tools
- **Vite** - быстрый сборщик и dev-сервер
- **ESLint** - линтинг кода для качества
- **Prettier** - автоформатирование кода
- **Jest** - unit тестирование
- **Testing Library** - тестирование React компонентов
- **Husky** - git hooks для качества коммитов

## 🔧 Установка и запуск

### Предварительные требования
- Node.js 18+
- npm или yarn

### Установка
```bash
# Клонирование репозитория
git clone <repository-url>
cd modsen-calendar

# Установка зависимостей
npm install

# Настройка переменных окружения
cp .env.example .env
# Заполните .env файл вашими Firebase credentials
```

### Запуск в режиме разработки
```bash
npm run dev
# Приложение будет доступно на http://localhost:3000
```

### Сборка для продакшена
```bash
npm run build
# Собранные файлы будут в папке dist/
```

### Тестирование
```bash
# Запуск всех тестов
npm test

# Запуск линтера
npm run lint

# Проверка типов TypeScript
npm run type-check
```


```bash
npm install -g firebase-tools
firebase init hosting
npm run build

```
